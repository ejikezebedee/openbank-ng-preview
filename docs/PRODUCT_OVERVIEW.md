# OpenBank NG Product Overview

## Positioning

OpenBank NG is a commercial source-code foundation for Nigerian wallet, neobank, cooperative finance, agency banking, and payment-product builds.

## Buyer Pain

Most fintech templates provide screens only. Real fintech teams need app surfaces, backend workflow boundaries, wallet/account records, money handling, KYC states, transfer controls, admin review, audit logs, provider integration boundaries, release documentation, and clear production caveats.

## Product Promise

OpenBank NG helps a buyer start from a serious fullstack fintech foundation instead of a blank repository.

## Core Surfaces

- Customer banking app.
- Admin operations dashboard.
- Backend API.
- Shared banking domain package.
- PostgreSQL-compatible schema.
- Buyer documentation and release materials.

## Nigerian Banking Orientation

- NGN/kobo money handling.
- Nigerian bank-code support.
- BVN/NIN-ready KYC fields.
- KYC tiering.
- NUBAN-style account-number support.
- NIP/NIBSS-style transaction status modeling.
- Provider adapter boundary for licensed integrations.

## Production Caveat

The product must be customized, audited, integrated with licensed providers, legally reviewed, and approved before live regulated use.
